# PDF to Audio Converter

## Description
This is a PDF to Audio converter made using Python as the base and Flask to make it a web application. The user can upload a pdf and the website loads and shows a preview of the entire file on-screen. 
By clicking just clicking one button, the website converts the pdf into audio using gTTS in the background. 
After the Audio is converted, a <audio> control is shown through which user can play and pause the audio, increase the playback speed and move to different parts of the audio. 

## Technologies used
- Python
- Flask
- gTTS
- HTML
- CSS
- JavaScript
